# Developer Certificate of Origin (DCO)

All contributions to FireClaw must include a sign-off indicating that the contributor agrees to the Developer Certificate of Origin.

## Why DCO?

FireClaw uses the AGPLv3 license and may offer dual licensing in the future. The DCO ensures that all contributions are properly licensed and that the project maintainer retains the ability to manage licensing for the project's sustainability.

## How to Sign Off

Add a `Signed-off-by` line to your commit messages:

```
git commit -s -m "Your commit message"
```

This adds a line like:

```
Signed-off-by: Your Name <your.email@example.com>
```
