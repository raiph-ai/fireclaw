API Endpoints:
  POST   /api/report                  Submit threat detection report
  GET    /api/blocklist               Retrieve flagged domains
  GET    /api/patterns                Retrieve threat pattern signatures
  GET    /api/stats                   Public dashboard statistics
  GET    /api/reputation/:domain      Domain reputation lookup
  GET    /health                      Health check

Database Tables:
  instances                           Registered FireClaw instances
  reports                             Individual threat detection reports
  domains                             Aggregated domain threat intelligence
  patterns                            Threat pattern signatures

Quick Start:
  npm install                         Install dependencies
  npm run init-db                     Initialize database
  node validate-setup.mjs             Validate configuration
  npm start                           Start server
  node test-client.mjs                Test API

Production:
  pm2 start server.mjs                Start with PM2
  sudo systemctl start fireclaw-api   Start with systemd
